Om webbplatsen

Syfte

Den här webbpapplikationen är en det av kursen Backend-utveckling DT207G på MIUN- Syftet med laborationen är att bygga en CRUD-applikation där både backend och frontend kommunicerar med varandra. Samt där backend är ett eget REST api. webbplatsen består av tre sidor:

  • Startsida - som visar data
  • Undersida för att lägga till data
  • Aboutsida som beskriver uppgiftens, syfte och slutsatser

Databas och server

Applikationen använder mongoDB som databas tillsammans med Mongooser för schemahantering. Backend är byggd med Node.js och Express kommunikationen med databasen vis Mongoose. Alla data hanteras genom ett eget REST api som som innehåller CRUD-funktionalitet som POST, GET, DELETE & PUT. Frontend och backend är två separata tjänster, där backend är publicerad separat. Frontendsidan är publicerad via Netlify.

Slutsats

Genom att arbeta med denna laboration har jag fått en tydligare förståelse hur noSQL fungerar och hur mongoDB skiljer sig från relatiaonsdatabaser. Jag har lärt mig hur man skapar och använder Mongoose-scheman för at strukturera och validera datan. Samt återigen arbeta med en fullstack applikation där man lägger upp och hantera REST api (backend) och frontend själv.
